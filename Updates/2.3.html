New features

Favicon support (2.2.4): icons of the website is displayed in Growl notifications and in front of each menu items.

Automatic update support will work from now on (≥ 2.3) using the new distribution site.
Jaeho Shin decided to fully fork Mike Godenzi and Claudio Marforio's Reader Notifier Reloaded project after observing lack of activity over the past few years. Beginning with version 2.3, it will check the new site for updates.

Bug fixes

Google Reader changed its feed preview URL a long time ago, and finally this is reflected in our code base. If you were experiencing problems opening RSS/Atom feed from Safari, it'll now show up correctly from Google Reader.
